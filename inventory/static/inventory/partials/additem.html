Add New Inventory Item

Before adding an Item, ensure that there isn't already one like this listed in the Items List.

If an equivalent Item already exists, you should add another Instance to the existing Item, rather than adding a new Item.

Name The title of this item (keep it short and to the point).
Details Any descriptive information. Should be more detailed than the name.
Category The type of equipment that this item is.
Owner Who actually owns this item?
Responsible Committee Position The member of the Cinema committee responsible for looking after the item.
Wiki Page The name of the Wiki page about this item (inside the Assets/* namespace), e.g. "Kinoton Projector".
